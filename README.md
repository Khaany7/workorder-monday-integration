# Workorder Monday Integration

A simple integration between work orders and Monday.com.

## Features

- Sync work orders with Monday.com boards
- Easy setup and configuration

## Getting Started

1. Clone the repository.
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Configure your Monday.com API credentials.
4. Run the integration:  
   ```bash
   npm start
   ```
